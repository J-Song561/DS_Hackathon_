Project Title - **'WatchWise'**

사용자가 시청하는 유튜브 영상을 감지하고 유해도를 판단해 주의를 주는 확장 프로그램입니다. 디지털 문해력 부족으로 인해 해로운 영향에 특히 노출되기 쉬운 청소년/노인층이 동영상을 시청할 때, 분별력을 보조할 수 있는 프로그램을 만들고자 했습니다. 사용자는 영상을 시청하면서 유해성 알림을 실시간으로 받을 수 있으며 팝업을 클릭해 상세분석 내용도 확인할 수 있습니다.

Technologies: HTML, CSS, JS, Json, python, django, MobileNetV2
